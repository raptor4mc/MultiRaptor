How this training works

Each assignment has its own dedicated page with a long explanation about exactly one coding topic. You will get a focused task, a small code editor, and a terminal output area under it. When your solution passes, the next assignment unlocks.

This flow trains you step-by-step until you can write the patterns confidently, then ends with a tougher debugging assignment where you repair broken code.
