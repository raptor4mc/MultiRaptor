# Contrib

This folder is for non-core assets that help the MagPhos ecosystem grow without adding churn to `src/`.

## Structure
- `community-modules/` – community-maintained modules/packages/snippets.
- `optional-tools/` – helper scripts and utilities that are useful but not required for core builds.
- `experimental/` – unstable experiments and prototypes.

## Guidance
- Keep production-critical runtime/compiler code in `src/`.
- Clearly document ownership and stability in each contributed item.
- Prefer small, focused contributions with a short README per subfolder.
