# contrib/

This workspace is for non-core additions so the language core stays stable and focused.

## Purpose

- **community-modules/**: shared modules maintained by the community.
- **optional-tools/**: helper tools that are useful but not required for core compilation/runtime.
- **experimental/**: prototypes and trial integrations that may or may not graduate into core.

Keep anything under `contrib/` decoupled from mandatory compiler/runtime paths unless and until it is promoted.
